geostat-rs in the browser

Fits a variogram model to 80 random sample points of a synthetic field and ordinary-kriges a 60×60 grid — entirely client-side via WebAssembly.

Build first: wasm-pack build crates/geostat-wasm --target web --release, then serve the repo root (python3 -m http.server) and open /examples/wasm-demo/.
